```math
I_{zz} = I_{xx} + I_{yy} = I_{mm} + I_{nn}
```

$ x $, $ y $, $ z $ are a set of axes. $ m $, $ n $, $ z $ are another set of
axes.

If $I_{xx}$ is at maximum, $I_{yy}$ will be at minimum.
